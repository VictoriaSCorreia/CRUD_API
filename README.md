## CRUD

Task Management API (to-do) :

* Create a new task with the fields : 

      1 - title
      2 - description
      3 - dueDate

* Read all the tasks list

* Update a task and any of its fields

* Delete a task

## MySQL usage

```bash
# run these lines on the query

$ ALTER USER 'root'@'localhost' IDENTIFIED WITH mysql_native_password BY '<password>';

$ flush privileges;
```

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
